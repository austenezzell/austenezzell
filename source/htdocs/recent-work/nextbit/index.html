Nextbit

We helped Nextbit introduce Robin, a forward-thinking android based smartphone that never runs out of space.

The website is currently on it's sixth iteration. We have been refreshing the site with teaser images, different messaging and glimpses into the phone's UI since they came out of stealth mode.

The website features an interactive cloud that represents Nextbits main differentiator - cloud storage. The cloud is generated at different sizes, shapes and speeds each time the website loads. We even enabled the accelerometer on mobile devices so you tilt and shake the cloud.
